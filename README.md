## Usage

1. Install the node modules

  use NPM :
  ```sh
    npm install
  ```

  OR use YARN :
  ```sh
    yarn install
  ```

2. Run the script ```dev:api``` to start development with local server side

  with NPM :
  ```sh
    npm run dev:api
  ```

  OR with YARN :
  ```sh
    yarn dev:api
  ```

3. Run the script ```build``` to make the compressed files

  with NPM :
  ```sh
    npm run build
  ```

  OR with YARN :
  ```sh
    yarn build
  ```
